# 期末專題主題

## 專題名稱

漁港天氣提醒 App（Fishing Harbor Weather Alert App）

## 主題方向

海事與海洋生活安全。

## 開發動機

台灣四面環海，許多民眾會前往漁港觀光、拍照、釣魚或搭船。漁港附近的風速、浪高與潮汐變化會影響安全，因此本專題希望以 Android App 整合漁港位置、天氣資訊、風浪警戒與最愛漁港功能，協助使用者快速判斷是否適合前往。

## 核心功能

1. 漁港清單與天氣摘要。
2. 詳細風速、浪高、降雨、潮汐資訊。
3. 安全等級：安全、注意、危險。
4. 最愛漁港與提醒條件設定。
5. Google Maps 顯示漁港位置與使用者 GPS。
6. Flask API 範例，可部署到 AWS EC2。
